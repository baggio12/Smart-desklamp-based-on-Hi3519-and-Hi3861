# Smart-desklamp-based-on-Hi3519-and-Hi3861
基于 **Hi3519** 与 **Hi3861** 的双芯智慧台灯系统。  
Hi3519 利用摄像头进行人体关键点识别与坐姿分析，Hi3861 负责环境光/色温传感器采集，并智能调节 LED 亮度和色温，从而在护眼的同时帮助矫正坐姿。

---

## 功能特性

- **坐姿检测**  
  通过 Hi3519 实时推理人体关键点（骨骼点），运行自研坐姿算法，判断前倾、侧歪、驼背等不良姿态，并给出提醒或联动灯光调整。
- **智能调光**  
  利用光强传感器（BH1750）与色温传感器（TCS34725），结合当前环境自动调节 LED 的 PWM 亮度与色温比例。
- **数据上报（可选）**  
  支持将坐姿时长、环境光曲线等上传至云端或本地服务器，便于家长或用户复盘。
